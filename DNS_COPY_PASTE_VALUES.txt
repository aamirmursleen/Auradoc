VERIFICATION CHECKLIST
=================================================================

After adding all records:

1. Wait 2-6 hours for DNS propagation

2. Check SPF:
   - Go to: https://mxtoolbox.com/spf.aspx
   - Enter: mamasign.com
   - Should show: v=spf1 include:_spf.resend.com ~all

3. Check DMARC:
   - Go to: https://mxtoolbox.com/dmarc.aspx
   - Enter: mamasign.com
   - Should show: v=DMARC1; p=quarantine...

4. Check DKIM:
   - Go to: https://mxtoolbox.com/dkim.aspx
   - Enter: resend._domainkey.mamasign.com
   - Should show: Public key details

5. Verify in Resend:
   - Go to: https://resend.com/domains
   - Click "Verify" next to mamasign.com
   - Status should turn green "Verified"
